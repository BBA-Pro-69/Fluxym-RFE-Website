📖 Exemple de fichier rempli
{
  "name":      "KREAMOB",
  "legalName": "KREAMOB Legal entity",
  "siren":     "333191120",            // 9 chiffres
  "nic":       "28662",                // optionnel, 5 chiffres
  "vatNumber": "FR12333191120",         // FR + 2 chiffres + SIREN
  "address": {
    "street":  "10 Avenue du Marechal Foch",
    "city":    "VILLEFRANCHE-SUR-MER",
    "zip":     "06230",
    "country": "FR"
  }
}

ℹ️ SIRET = SIREN (9 chiffres) + NIC (5 chiffres). Le NIC est optionnel : s’il est absent, la valeur 00001 sera utilisée par défaut.
